# Security Policy

## Research and Educational Implementation

**CRITICAL: This repository is a research template and contains proof-of-concept cryptographic implementations intended for academic use and educational purposes only.**

As an academic framework, the implementations within this repository have not undergone formal security audits, third-party cryptographic peer review, or hardening against side-channel attacks (e.g., timing analysis).

## Production Use Warning

- **DO NOT** use the code produced with this template in production environments.
- **DO NOT** use this code to protect high-value assets, sensitive data, or financial transactions.
- **DO NOT** assume constant-time execution unless explicitly implemented and verified with appropriate tools.

## Reporting Vulnerabilities

If you identify an implementation-level flaw that affects the cryptographic correctness of the template's baseline primitives, please open a GitHub Issue. However, the context of this project as an experimental research platform remains the priority.
